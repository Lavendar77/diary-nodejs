# Languages
Built with ❤️ using:
- Node.js
- TypeScript
- Express
- Serverless Framework

## Development
- Installation: `npm install`

### Express App
- Compile the TypeScript modules: `npm run build`
- Run the app: `npm run start`

### Serverless
- Run the offline version: `npm run serverless`
